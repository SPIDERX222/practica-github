## Documentacion
Pregunta 1

GitHub es una plataforma para guardar y colaborar en código, 
y se basa en Git, que es el sistema que gestiona las versiones de ese código.

Pregunta 2

Repositorios públicos, que pueden ser vistos por cualquier persona y 
son ideales para proyectos abiertos o colaborativos.

Repositorios privados, que solo son accesibles para los usuarios autorizados
y se usan para trabajo interno, proyectos personales o código
 que no se desea compartir públicamente.
